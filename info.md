# Solar Energy Optimizer

## ⚠️ DO NOT USE - Under Development

**WARNING**: This integration is under active development and should not be used at this time.

The code is not yet functional and has not been tested. Do not install this integration.

---

Automate your home energy management by optimizing battery charge/discharge cycles based on:
- Dynamic electricity prices (Frank Energie)
- Solar production forecasts (Solcast)
- Your battery status (Solax Modbus)

## Quick Start

1. Ensure you have these integrations installed:
   - Solax Modbus
   - Solcast Solar
   - Frank Energie

2. Install this integration via HACS

3. Add the integration and configure your entity selections

4. Choose your optimization strategy and let it run!

## Features

- 4 optimization strategies (minimize cost, maximize self-consumption, grid independence, balanced)
- Real-time monitoring and cost tracking
- Manual override capability
- Service for on-demand optimization

For detailed documentation, see the [README](https://github.com/xlith/ha-solar-energy-optimizer).
